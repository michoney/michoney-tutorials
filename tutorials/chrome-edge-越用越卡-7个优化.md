# 为什么 Chrome / Edge 会越来越卡？
在优化之前，你需要先理解原因：
### 核心原因有3个：
- 浏览器功能越来越复杂（GPU加速、插件等）
- 后台进程过多，占用CPU和内存
- 长期使用产生缓存堆积
本质：**资源被浪费，而不是性能不够**
# ✅ 方法一：关闭硬件加速（最关键优化）
### 操作步骤
1. 打开浏览器设置
1. 进入「系统」
1. 关闭「使用硬件加速（GPU）」
1. 重启浏览器
### 为什么有效？
硬件加速会：
- 把任务交给 GPU
- 在多标签 + 插件情况下导致冲突
- 出现卡顿、掉帧
✔ 关闭后：
👉 使用更稳定的 CPU 渲染
👉**流畅度明显提升**
# ✅ 方法二：禁用 GPU 栅格化和 2D 加速
### 操作方法
地址栏输入：

- GPU Rasterization
- Accelerated 2D Canvas

### 优化效果
- 页面滚动更顺滑
- 视频播放更稳定
- 减少渲染延迟
👉 特别适合：低配电脑 / 多任务用户
# ✅ 方法三：删除无用浏览器插件
### 为什么插件会拖慢浏览器？
每一个插件：
- 都在后台运行
- 占用内存 + CPU
- 可能产生冲突
👉 插件越多 = 浏览器越慢
### 优化建议
- 删除长期不用的插件
- 只保留必要工具
- 避免重复功能插件
✔ 效果：**立竿见影**
# ✅ 方法四：开启标签页节能（Memory Saver）
### Chrome 设置
设置 → 性能 → 开启「节省内存」
建议：选择「均衡模式」
### Edge 设置
设置 → 系统和性能 → 开启「睡眠标签页」
建议：
- 30秒自动休眠
- 开启标签淡化
### 优势
- 自动释放后台标签内存
- 多开网页依然流畅
**非常适合重度用户**
# ✅ 方法五：关闭后台运行（隐藏性能杀手）
### 操作路径
设置 → 系统 → 关闭：
**“关闭浏览器后继续运行后台应用”**
### 为什么要关？
很多用户不知道：
**浏览器关闭后仍在运行**
这会导致：
- CPU持续占用
- 内存无法释放
- 系统变慢
✔ 关闭后：电脑明显更流畅

### Q1：关闭硬件加速会影响性能吗？
不会。
在大多数电脑上，关闭后反而更稳定、更流畅。

因为：
- 缓存堆积
- 插件增多
- 后台进程变多
